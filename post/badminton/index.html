招式

杀球

杀斜线身体最好朝向对角线方向

跳杀时是略向前跳

杀球时要想象在是比赛

右脚的势能转移到左脚上(并步运动)

高远

向上将球推出的感觉

击球点头顶正上方

球拍接触球前所积蓄的力量，是打远距离和高强度的重要因素

反手

运动同时加入扭转，一边蓄力一边移动

  • 练习

    反手最大限度扔羽毛球

平高球

在对方已经站好接杀位置时压对方到后场

吊球

摆好准备姿势比杀球和高远球更重要

劈吊

  1. 动作和打杀球一样
  2. 在学好吊对角线后再来学

挑球

  1. 脚尖对准来球方向
  2. 躯干与跨步的稳定性更重要
    1. 挺胸直背
    2. 迈步时躯干的强度及骨盆的灵活性

正手

球拍由下向上挥非常重要

反手

  1. 握拍时，拇指一定要坚起来，挥拍感觉由下向上扫
  2. 击球时，感觉是拇指将球推出

放网前球

  1. 球拍向身体正前方抬起
  2. 球拍高度与网上面的白布条一致
  3. 球拍向上托的感觉
  4. 反手放网前球比正手击球时要略低，击球前最好左脚后蹬来提高击球高度

推球

  1. 球拍向上举起，向网前移动
  2. 击球时蹬地起跳，出拍迎球
  3. 只靠手腕力量击球的感觉
  4. 击球后身体不要过渡前倾，为下一拍做准备
  5. 双打比赛前场推球不能直接得分要做好多拍准备

接发球

  1. 应对对方强杀
  2. 重点：拍面与来球方向一致
  3. 接发球前骨盆前屈，是强有力接球的主要原因
  4. 从身体发力
  5. 小指的练习非常重要

正手

  1. 重心在左脚拇指上，不要抬脚后跟
  2. 借力打力
  3. 在拍面与球网平行的位置击球
  4. 身体重心迅速跟进
  5. 在身体的正面击球
  6. 利用膝盖的屈伸力量

反手

  1. 向后撤拍
  2. 球和身体的距离不长于一个球拍的长度就用反手
  3. 在拍面与球网平行的位置击球
  4. 身体重心迅速跟进
  5. 要大幅度挥拍，尽可能往远打

建议

接发球不能形成对打，则可以放慢球速

接杀直线与斜线

  1. 对方杀直线时，在身体的侧方也可以接球
  2. 对方杀斜线球要在身体前方接发球

平抽球

正手

  1. 拍头向上，向球的方向出脚
  2. 在身体前方一点的位置击球
  3. 对于慢的平抽球要大幅度挥拍，快的球要缩短挥拍路径

反手

  1. 向来球方向跨步
  2. 要在身体前方击球
  3. 在大幅度挥拍中找到击球点
  4. 找到将球推出去的感觉
  5. 击球时蹬地是关键

发球

发短球

击球时感觉是将球推出

发长远球

要用到腰部力量

右半场防守反击

反手腰位球处理

单打

  1. 打直线球非常重要
  2. 重要关注打6点(防守过渡也是)：4个角和中场两侧边线
  3. 启动慢了可以单脚蹬地提高移动速度

双打

  1. 时刻准备启动
  2. 时刻准备接球的意识培养
  3. 行思考再击球
  4. 前场球员要有节奏的小幅度摆动

多球

  1. 先要瞄准落点准确击球，再加快速度
  2. 姿势摆好就进攻，没摆好就确保击球线路正确

步伐

并步和跳步等有变化的步伐是支持身体重心移动的重要技术

右前

  1. 起动跳跃重心放在左脚上
  2. 右脚向外跨左脚跟上
  3. 右脚向外蹬做弓箭步

躯干训练

需要关注髋关节、肩关节及包含骨盆在内的脊柱一起进行。清晰感受这些部位的动作、明确其位置的同时进行躯干训练是十分重要的，也是取得进步的捷径
