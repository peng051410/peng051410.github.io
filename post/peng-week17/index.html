Programming On 34 Keys · oppi.li

作者自己构建了一个分离式的34键的键盘用来日常编程，由于只有34个键，包含26个字母+4个常用符号+4个功能键(Tab，Space, Enter，Backspace)，对于其它的功能键就需要使用不同的layter来完成，作者就定义了三种模式来实现不能的功能

  1. NAV层，导航模式，实现上下左右的移动，通过按住左侧的 Space 触发
  2. SYM层，符号模式，包含所有常用的符号，主要集中在左侧键盘，通过按住右侧的 Enter 触发
  3. NUM层，数字模式，包含所有数字，主要集中在右侧键盘 ，通过按住左侧的 Tab 触发

对于常用的 Esc 键盘，作者通过组合按键的方式来实现，分别按住左右两侧的键

梵蒂冈城

  1. 梵蒂冈，官方名称为梵蒂冈城国，一个 内陆主权国家和城邦 。它被意大利首都罗马包围，历史上曾是罗马的一部分。它于1929年通过拉特兰条约从意大利王国独立。它由圣座治理，圣座本身是一个根据国际法的主权实体，保持其世俗权力、治理、外交和精神独立。梵蒂冈也用作教皇、圣座和罗马公教廷的代名词。
  2. 面积为49公顷（121英亩） ，2024年人口约为882人 ，它是 世界上面积和人口最小的主权国家 。它是世界上人口最少的首都之一。作为教廷的管辖，梵蒂冈城国是一个由教皇统治的教会或神职-君主制国家，教皇是罗马主教和天主教会的首脑。最高国家官员都是来自不同来源的天主教神职人员。在阿维农教皇时期（1309–1377）之后，教皇主要居住在如今的梵蒂冈城内的宗座宫，尽管有时也居住在罗马的奎里纳尔宫或其他地方。
  3. 圣座可以追溯到早期基督教，是天主教会的主要主教席位。到2018年，全球约有13.29亿受洗的天主教徒，分布在拉丁教会和23个东部天主教教会。独立的梵蒂冈城国家于1929年通过圣座与意大利王国之间的拉特兰条约成立，该条约将其视为一种新创造，而不是之前涵盖了中意大利大部分地区的更大教皇国（756-1870）的遗迹。
  4. 梵蒂冈城包含宗教和文化遗址，如圣彼得大教堂、西斯廷礼拜堂、梵蒂冈宗座图书馆和梵蒂冈博物馆。它们展示了世界上一些最著名的画作和雕塑。梵蒂冈城的经济主要依靠天主教信徒的捐款、邮票和纪念品的销售、博物馆的入场费以及出版物的销售来支持。 梵蒂冈城没有税收，物品免税 。
