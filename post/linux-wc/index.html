前言

在日常应用中，统计文件内容是一种常见的需求，可以通过 wc 命令来实现

Wc

wc file

统计文件，显示行数、单词数、字节数，带文件名

wc -c file

统计文件，只显示字节数

wc -w file

统计文件，只显示单词数

wc -l file

统计文件，只显示文件行数

wc file1 file2

统计多个文件，显示汇总信息

wc < file

统计文件，显示行数、单词数、字节数，不带文件，便于shell脚本后续处理

wc -L file

打印出文件行最长的节数
